PowerSchool Svelte Web Component Integration

How to use this component in PowerSchool

This Svelte application is packaged as a web component that can be safely embedded in PowerSchool pages without style or JavaScript conflicts.

Integration Code

To add this component to a PowerSchool page, use the following code:

<!-- 1. Add the component tag where you want it to appear --> <ps-svelte-app></ps-svelte-app> <!-- 2. Load the component script (adjust the path as needed) --> <script type="module" src="/ps-svelte-template/index.js"></script>

Key Benefits

  • Style Isolation: The component uses Shadow DOM to isolate its styles from PowerSchool's styles
  • JavaScript Isolation: The component's JavaScript won't conflict with PowerSchool's JavaScript
  • Easy Integration: Just add the tag and load the script - no complex setup required
  • Maintainable: The component can be updated independently of the PowerSchool page

Troubleshooting

If the component doesn't appear:

  1. Check that the script path is correct
  2. Look for JavaScript errors in the browser console
  3. Verify that the plugin is properly installed in PowerSchool
