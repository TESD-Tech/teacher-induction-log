PowerSchool Svelte Components Demo

This page demonstrates how to use both the full Svelte application and individual components within PowerSchool.

1. Full Application

The complete Svelte application encapsulated as a custom element:

<!-- How to use the full application --> <ps-svelte-app></ps-svelte-app>

2. Individual Components

Individual Svelte components can be used independently:

Counter Component

<!-- How to use the Counter component --> <svelte-counter></svelte-counter>

Counter with Initial Value

<!-- How to use the Counter component with props --> <svelte-counter initial-count="10"></svelte-counter>

Integration Instructions

To use these components in your PowerSchool pages:

  1. Include the main JavaScript bundle:
    <script type="module" src="/ps-svelte-template/index.js"></script>
  2. Add the component tag where you want it to appear:
    <!-- For the full application --> <ps-svelte-app></ps-svelte-app> <!-- For individual components --> <svelte-counter></svelte-counter>
